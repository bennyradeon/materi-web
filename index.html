1. Struktur Dasar HTML

Konsep: Setiap halaman HTML harus memiliki deklarasi <!DOCTYPE html> dan elemen <html> sebagai root. Di dalamnya terdapat <head> (metadata) dan <body> (konten yang tampil).

<!DOCTYPE html>                 <!-- 1. Deklarasi tipe dokumen HTML5 -->
<html lang="id">                <!-- 2. Elemen root, atribut lang menentukan bahasa -->
<head>                           <!-- 3. Bagian kepala, berisi metadata -->
    <meta charset="UTF-8">      <!-- 4. Mengatur karakter encoding UTF-8 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 5. Agar tampilan responsif di HP -->
    <title>Portofolio Benny S.Kom</title> <!-- 6. Judul halaman di tab browser -->
</head>
<body>                           <!-- 7. Semua konten yang tampil ditaruh di sini -->
    <h1>Benny S.Kom</h1>        <!-- 8. Heading level 1, judul utama -->
    <p>Selamat datang di portofolio saya. Saya seorang pengajar dan pengembang web.</p> <!-- 9. Paragraf -->
</body>
</html>

<html lang="id">Elemen root (pembungkus) seluruh halaman. Atribut lang menentukan bahasa konten, berguna untuk mesin pencari dan pembaca layar.

<meta charset="UTF-8">Menentukan karakter encoding UTF-8, mendukung berbagai huruf dan simbol (termasuk emoji).

<title>...</title>Judul halaman yang muncul di tab browser dan hasil pencarian.
<body>Bagian yang berisi semua konten yang akan ditampilkan kepada pengguna (teks, gambar, video, dll).
<h1>...</h1>Heading level 1, biasanya digunakan untuk judul utama halaman. Ada 6 level heading (h1 sampai h6).
<p>...</p>Paragraf. Membuat blok teks dengan spasi sebelum dan sesudahnya.

2. Heading, Paragraf, Format Teks

Konsep: Selain <h1> dan <p>, ada tag untuk format teks seperti <strong> (penting), <em> (penekanan), <mark> (stabilo), dll. Tag <br> untuk baris baru, <hr> untuk garis horizontal.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portofolio Benny S.Kom</title>
</head>
<body>
    <h1>Benny S.Kom</h1>                                <!-- Heading utama -->
    <p>Selamat datang di portofolio saya.</p>           <!-- Paragraf biasa -->
    <hr>                                                  <!-- Garis horizontal pemisah -->

    <h2>Tentang Saya</h2>                                 <!-- Subheading (level 2) -->
    <p>Saya memiliki minat dalam <strong>pengembangan web</strong> dan <em>desain antarmuka</em>.</p>
    <!-- strong: teks tebal dengan makna penting, em: teks miring dengan penekanan -->

    <p>Saya juga suka <mark>coding</mark> dan <small>berbagi ilmu</small>.</p>
    <!-- mark: latar kuning seperti stabilo, small: teks lebih kecil -->

    <p>Hobi saya: <ins>membaca</ins> dan <del>bermain game</del> (sekarang lebih suka ngoding).</p>
    <!-- ins: teks disisipkan (garis bawah), del: teks dihapus (coret) -->

    <p>Rumus favorit: E = mc<sup>2</sup> dan H<sub>2</sub>O.</p>
    <!-- sup: superskrip (di atas), sub: subskrip (di bawah) -->
</body>
</html>

TagPenjelasan
<hr>Horizontal rule, membuat garis pemisah horizontal. Tag tunggal (tidak perlu penutup).
<h2>...</h2>Heading level 2, subjudul.
<strong>...</strong>Menandai teks yang penting. Biasanya ditampilkan tebal, dan memiliki makna lebih dari sekadar tampilan.
<em>...</em>Menandai teks yang ditekankan (emphasized). Biasanya ditampilkan miring.
<mark>...</mark>Menyorot teks seperti dengan stabilo. Tampilan default latar kuning.
<small>...</small>Membuat teks lebih kecil dari ukuran normal. Biasanya untuk catatan kaki atau teks sampingan.
<ins>...</ins>Menandai teks yang disisipkan (inserted). Biasanya ditampilkan dengan garis bawah.
<del>...</del>Menandai teks yang dihapus (deleted). Biasanya ditampilkan dengan garis tengah (coret).
<sup>...</sup>Superscript, teks muncul di atas garis normal. Contoh: pangkat.
<sub>...</sub>Subscript, teks muncul di bawah garis normal. Contoh: indeks kimia.

3. Link & Gambar

Konsep: <a> digunakan untuk tautan (hyperlink). Atribut href menentukan tujuan, target="_blank" membuka di tab baru. <img> untuk menampilkan gambar dengan atribut src (sumber), alt (teks alternatif), width/height (ukuran).

üìÅ Catatan: Pada contoh ini, gambar menggunakan file foto.jpg yang harus berada di folder yang sama dengan file HTML. Gantilah dengan file gambar Anda sendiri. 
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portofolio Benny S.Kom</title>
</head>
<body>
    <h1>Benny S.Kom</h1>
    <p>Selamat datang di portofolio saya.</p>
    <hr>

    <h2>Tentang Saya</h2>
    <p>Saya memiliki minat dalam <strong>pengembangan web</strong> dan <em>desain antarmuka</em>.</p>
    <p>Hobi saya: <ins>membaca</ins> dan <del>bermain game</del>.</p>

    <hr>
    <h2>Foto Profil</h2>
    <!-- Gambar lokal: pastikan file foto.jpg ada di folder yang sama -->
    <img src="foto.jpg" alt="Foto Benny S.Kom" width="300" height="300">

    <!-- Tautan ke GitHub, buka tab baru -->
    <p>Kunjungi <a href="https://github.com" target="_blank">GitHub saya</a> untuk melihat proyek.</p>
</body>
</html>

target="_blank"Menentukan di mana tautan dibuka. _blank membuka di tab/jendela baru. Tanpa ini, tautan dibuka di tab yang sama.

    <title>Portofolio Benny S.Kom</title>

    <h1>Benny S.Kom</h1>
    <p>Selamat datang di portofolio saya.</p>
    <hr>

    <h2>Tentang Saya</h2>
    <p>Saya seorang pengajar dan pengembang web.</p>
    <hr>

    <h2>Foto Profil</h2>
    <img src="foto.jpg" alt="Foto Benny" width="200">

    <h2>Riwayat Pendidikan</h2>
    <table border="1" cellpadding="8">
        <tr>
            <th>Tahun</th>
            <th>Institusi</th>
            <th>Gelar</th>
        </tr>

            <td>2015-2019</td>

            <td>S.Kom</td>
        </tr>
        <tr>
            <td>2019-2021</td>

5. HTML Semantik

Konsep: Tag semantik memberi makna pada struktur halaman, seperti <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>. Ini membantu SEO dan aksesibilitas.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portofolio Benny S.Kom</title>
</head>
<body>
    <header>
        <h1>Benny S.Kom</h1>
        <p>Pengajar & Pengembang Web</p>
    </header>

    <nav>
        <a href="#about">Tentang</a> |
        <a href="#projects">Proyek</a> |
        <a href="#contact">Kontak</a>
    </nav>

    <main>
        <section id="about">
            <h2>Tentang Saya</h2>
            <p>Saya memiliki pengalaman dalam mengajar dan mengembangkan web.</p>
        </section>

        <section id="projects">
            <h2>Proyek Saya</h2>
            <img src="foto.jpg" alt="Proyek" width="200">
            <p>Contoh proyek yang pernah saya buat.</p>
        </section>

        <section id="education">
            <h2>Pendidikan</h2>
            <table border="1">
                <tr><th>Tahun</th><th>Institusi</th></tr>
                <tr><td>2015-2019</td><td>Universitas Indonesia</td></tr>
            </table>
        </section>

        <section id="contact">
            <h2>Hubungi Saya</h2>
            <form>
                <label>Nama: <input type="text"></label><br>
                <label>Email: <input type="email"></label><br>
                <input type="submit">
            </form>
        </section>
    </main>

    <footer>
        <p>&copy; 2025 Benny S.Kom</p>
    </footer>
</body>
</html>

7. Selektor CSS (Class, ID, dll.)

Konsep: Selektor class (.) dan ID (#) digunakan untuk menarget elemen spesifik. Selektor atribut, descendant, child, dan universal juga penting.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portofolio Benny S.Kom</title>
    <style>
        /* Selektor class */
        .highlight {
            background-color: #ffeb3b;
            padding: 2px 5px;
            border-radius: 3px;
        }
        /* Selektor id */
        #projects {
            border-left: 5px solid #3b82f6;
            padding-left: 15px;
        }
        /* Selektor atribut */
        input[type="text"], input[type="email"] {
            border: 2px solid #ccc;
            border-radius: 4px;
            padding: 8px;
            width: 100%;
            max-width: 400px;
        }
        input[type="submit"] {
            background: #3b82f6;
            color: white;
            border: none;
            padding: 10px 20px;
            border-radius: 5px;
            cursor: pointer;
        }
        input[type="submit"]:hover {
            background: #2563eb;
        }
        /* Selektor descendant */
        section p {
            line-height: 1.6;
        }
        /* Selektor child */
        div > p {
            color: green;
        }
        /* Selektor universal */
        * {
            box-sizing: border-box;
        }
    </style>
</head>
<body>
    <header>
        <h1>Benny S.Kom</h1>
    </header>
    <nav>
        <a href="#about">Tentang</a>
        <a href="#projects">Proyek</a>
    </nav>
    <main>
        <section id="about">
            <h2>Tentang Saya</h2>
            <p>Saya suka <span class="highlight">coding</span> dan mengajar.</p>
        </section>
        <section id="projects">
            <h2>Proyek Saya</h2>
            <img src="foto.jpg" alt="Proyek" width="200">
        </section>
        <section id="contact">
            <h2>Hubungi Saya</h2>
            <form>
                <label>Nama: <input type="text" placeholder="Nama"></label><br>
                <label>Email: <input type="email" placeholder="Email"></label><br>
                <input type="submit" value="Kirim">
            </form>
        </section>
    </main>
    <footer>
        <p>Hak Cipta</p>
    </footer>
</body>
</html>

8. Box Model (Margin, Padding, Border)

Konsep: Setiap elemen HTML dianggap sebagai kotak yang terdiri dari konten, padding, border, dan margin. Aturan penulisan margin/padding bisa 1-4 nilai.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portofolio Benny S.Kom</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }
        body {
            background: #f0f0f0;
            padding: 20px;
        }
        .container {
            max-width: 800px;
            margin: 0 auto;
        }
        .box {
            width: 200px;
            height: 100px;
            background: #3498db;
            color: white;
            text-align: center;
            line-height: 100px; /* untuk vertikal tengah */
            margin: 20px;        /* margin semua sisi 20px */
            padding: 15px;       /* padding semua sisi 15px */
            border: 5px solid #2c3e50;
        }
        .box2 {
            width: 200px;
            height: 100px;
            background: #e74c3c;
            margin: 10px 20px;    /* atas-bawah 10px, kiri-kanan 20px */
            padding: 5px 10px 15px; /* atas 5px, kanan-kiri 10px, bawah 15px */
            border: 3px dashed #fff;
            border-radius: 15px;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="box">Kotak 1</div>
        <div class="box2">Kotak 2</div>
    </div>
</body>
</html>

9. Warna, Background, Gradient, Shadow

Konsep: Warna bisa dinyatakan dengan nama, hex, rgb, rgba, hsl. Background bisa warna polos, gambar, atau gradien. Shadow memberi efek bayangan pada teks atau kotak.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portofolio Benny S.Kom</title>
    <style>
        body {
            background: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 20px;
        }
        .container {
            max-width: 800px;
            margin: 0 auto;
        }
        header {
            background: linear-gradient(45deg, #667eea 0%, #764ba2 100%);
            color: white;
            padding: 40px;
            text-align: center;
            border-radius: 15px;
            box-shadow: 0 10px 20px rgba(0,0,0,0.2);
        }
        header h1 {
            text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
        }
        .card {
            background: rgba(255,255,255,0.9);
            padding: 20px;
